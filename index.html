Yes. GuildPass connects to your Stripe account via Stripe Connect.

Can I test before going live?

Yes, you can run everything in Stripe test mode first.

Does GuildPass read my DMs?

No. GuildPass only manages roles in your server using Discord's permissions.

Can I use multiple Discord servers?

Yes, you can connect and manage multiple servers from the dashboard.
